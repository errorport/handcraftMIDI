## Compiling

The project uses CMake and it is still configured.


1.

```sh
$ cd handcraftMIDI
```

2.

```sh
$ cmake .
```

3.

```sh
$ make
```
4.
```sh
$ ./handmarker
```

___ 

## Requirements


- OpenCV 3.3.0-rc
- G++ 6.3.0
- cmake 3.7.2
- GNU Make 4.1
